# WEB_Diary
Spring을 사용해 일기장 만들어보기



### 기능

* 회원가입
* 로그인
* 일기 목록(게시판)
* 일기 쓰기(텍스트/사진 삽입-사진 위치는 고정)
  * 날짜/날씨/기분 입력
  * 배경 색상 설정
  * 글자 크기 11pt 고정
  * 공개범위 설정
* 일기 삭제
  * 삭제 후 휴지통에서 3일간 보관 후 완전 삭제 
* 일기 댓글/공감
* 친구 추가, 삭제(차단)
  * 이웃, 서로이웃
